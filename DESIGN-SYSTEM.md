# JobSeeker Pro — Apple HIG Design System

## Design Philosophy
- **Clarity**: Content is king. UI recedes, information leads.
- **Deference**: Fluid motion, depth through translucency, not decoration.
- **Depth**: Layered interfaces with clear spatial hierarchy.

## 8 Core Screens
1. Landing (marketing)
2. Dashboard (home hub)
3. Job Search (browse + detail)
4. Resume Optimizer (audit + generate)
5. Mock Interview (voice AI)
6. Application Tracker (kanban)
7. Analytics (data viz)
8. Profile (settings)

## Color System
- Background: #000000 → #0a1020 (true black to deep navy)
- Surface 1: rgba(255,255,255,0.04) — cards
- Surface 2: rgba(255,255,255,0.08) — elevated cards
- Surface 3: rgba(255,255,255,0.12) — active states
- Accent: #0A84FF (iOS system blue)
- Success: #30D158
- Warning: #FFD60A
- Destructive: #FF453A
- Text Primary: #FFFFFF
- Text Secondary: rgba(255,255,255,0.6)
- Text Tertiary: rgba(255,255,255,0.3)
- Separator: rgba(255,255,255,0.08)

## Typography (SF Pro inspired via system font)
- Large Title: 34px/700
- Title 1: 28px/700
- Title 2: 22px/700
- Title 3: 20px/600
- Headline: 17px/600
- Body: 17px/400
- Callout: 16px/400
- Subhead: 15px/400
- Footnote: 13px/400
- Caption: 12px/400

## Spacing
- 4px grid system
- Content padding: 20px (mobile), 32px (desktop)
- Card padding: 16px
- Section gap: 32px
- Element gap: 12px

## Components
- Cards: 16px radius, surface-1 bg, 1px separator border
- Buttons: 12px radius, 50px height (large), 36px height (small)
- Inputs: 12px radius, 44px height (HIG minimum touch target)
- Badges: 8px radius, 24px height
- Navigation: segmented control style tabs
